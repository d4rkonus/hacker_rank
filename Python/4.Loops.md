```python
numbers = int(input())

for number in range(numbers):
print(number ** 2)
```

En este código se lee un número entero **numbers** de la entrada. Luego, con *range(n)*  se generan los números dentro del rango. El bucle **for** recorre esos números y, en cada vuelta, se imprime el cuadrado del número actual usando ***number ** 2*** .
