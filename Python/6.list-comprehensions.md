```python
if __name__ == '__main__':
	x = int(input())
	y = int(input())
	z = int(input())
	n = int(input())

result = [[i, j, k]
	for i in range(x + 1)
	for j in range(y + 1)
	for k in range(z + 1)
	if i + j + k != n]

print(result)
```

Este código es una combinación entre listas y bucles, en primer lugar la función **if __name__ == '__main__':** revisa si el script está siendo ejecutado directamente o es importado como un módulo adicional. 

Después se encuentran **4 variables (x, y, z, n)** que esperan números enteros como entradas, y en la otra mitad es donde se combinan las listas con los bucles. 

Se recorren **3 bucles anidados** uno para *i*, uno para *j* y otro para *k*, se crea una lista **[i, j, k]**.   

Si la suma de estos tres valores no es igual a *n*, la combinación se guarda en la lista **result**.
