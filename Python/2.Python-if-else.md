```python
#!/bin/python3

n = int(input().strip())

if n % 2 == 1:
	print("Weird")
elif n % 2 == 0 and n >= 2 and n <= 5:
	print("Not Weird")
elif n % 2 == 0 and n >= 6 and n <= 20:
	print("Weird")
else:
	print("Not Weird")
```

Este código evalúa el número que recibe como entrada mediante varias condiciones. Primero comprueba si el número es impar; en ese caso, muestra el mensaje **"Weird"**. Si no lo es, verifica si el número es par y está entre 2 y 5; de cumplirse, muestra **"Not Weird"**. Luego comprueba si el número es par y está entre 6 y 20; en ese caso, vuelve a mostrar **"Weird"**. Por último, si ninguna de las condiciones anteriores se cumple (es decir, si el número es par y mayor que 20), muestra el mensaje **"Not Weird"**.
