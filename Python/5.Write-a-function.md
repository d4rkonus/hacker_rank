```python
def is_leap(year):
	leap = False 
	
	if year % 400 == 0:
		leap = True
	elif year % 100 == 0:
		leap = False
	elif year % 4 == 0:
		leap = True
		
	return leap


year = int(input())
print(is_leap(year))
```

En este código se define una función llamada **is_leap** que es la función que recibe el año como un número y de primeras asume que el año que reciba **NO** es bisiesto por la condición de **leap = False**. 

Después empiezan las condiciones para definir si el número es bisiesto o no.

-  Primero esta condición define que si el resto de dividir el año entre ***400*** es 0, significa que el año es bisiesto: 

```python
if year % 400 == 0:
	leap = True
```

-  Si la condición anterior **NO** se cumple, esta siguiente condición entra en pista, revisa si el año es divisible entre ***100***, si se cumple, el año **NO** es bisiesto :

```python
elif year % 100 == 0:
	leap = False
```

-  Y si ninguna de las condiciones anteriores se cumple, esta siguiente condición entra en pista, revisa si el año es divisible entre ***4***, si se cumple, el año es bisiesto:

```python
elif year % 4 == 0:
	leap = True
```

-  Para definir que el año es una entrada al código definida como número entero, se crea esta variable 

```python
year = int(input())
```

-    Y al final de la ejecución del programa, esta función muestra el resultado por pantalla: 

```python
print(is_leap(year))
```
